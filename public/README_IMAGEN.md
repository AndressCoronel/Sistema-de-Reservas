# Instrucciones para Agregar la Imagen de Fondo

## Paso 1: Agregar la Imagen

1. Coloca la imagen de fondo de la barbería en la carpeta `public/`
2. Nombra el archivo: `barberia-background.jpg` (o `.png`)

## Paso 2: Actualizar el Código (Opcional)

Si quieres usar un nombre diferente o formato diferente, edita el archivo:
- `app/page.tsx`
- Busca la línea: `backgroundImage: 'url(/barberia-background.jpg)'`
- Cambia el nombre del archivo según corresponda

## Formatos Recomendados

- **Formato**: JPG o PNG
- **Tamaño**: 1920x1080px o superior
- **Peso**: Menos de 500KB (optimizado para web)
- **Contenido**: Foto de la barbería, herramientas, ambiente, etc.

## Nota

Si no agregas una imagen, se mostrará un fondo degradado oscuro como respaldo.
